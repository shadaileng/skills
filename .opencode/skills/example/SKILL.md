# 示例 Skill

这是一个示例 skill，用于展示 opencode skill 的标准结构和格式。

## 功能说明

- 展示 skill 文件的基本结构
- 演示 frontmatter 格式
- 提供创建新 skill 的模板

## 使用方法

1. 复制此目录结构
2. 修改 `SKILL.md` 文件内容
3. 更新 frontmatter 中的 `name` 和 `description`

## 文件结构

```
.opencode/skills/
└── example/
    └── SKILL.md
```

## 注意事项

- `name` 必须与目录名一致
- `description` 应该清晰描述 skill 的用途和触发条件
- 使用 markdown 格式编写 skill 内容
